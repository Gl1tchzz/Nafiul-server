# Nafiul Server

A self-hosted Raspberry Pi homelab used to host and manage multiple applications.

## Features

- Docker Compose
- Homepage Dashboard
- Portainer
- NAS
- GitHub CI/CD
- Automatic Deployments
- Monitoring
- Backup Scripts

## Hosted Projects

- MacBook Tracker
- AI Image Grader (coming soon)
- Personal Website (coming soon)

## Hardware

- Raspberry Pi
- Ubuntu Server
- 1TB SSD
